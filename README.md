# Goblin Fashion Engine API

Backend service for the Goblin Fashion Engine wardrobe system.

## Tech Stack

- Java 21
- Spring Boot
- Firebase Authentication
- Firestore
- Google Cloud Run

## Responsibilities

- Canonical domain models
- Wardrobe persistence
- Outfit generation engine
- Rule evaluation (Quirks)
- OpenAI integration
